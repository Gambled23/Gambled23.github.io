The parking zone

A system for parking-spaces administration, auto-assigning the nearest available space, and printing a QR code with the assigned space.

Made entirely using python, this project uses tkinter to give the admin an easy to use GUI
As an administrator, you can watch live status of the parking lot, receive notifications from cars parked for more than 48 hours, create and delete parking spaces, and everything you would expect from a parking lot administration software.

The software runs entirely on the local PC using a self-hosted postgres database, but thanks to the GUI the administrator user doesn't need to know anything about databases to fully use The parking zone.

(special thanks to sami-sopas for the css ticket art)
